# 背景与目标

## 背景

目前所有研发素材、用户手册、PPT、视频等资料全部存放于 Wiki，内容庞杂散乱，不易查找。

Wiki 简单易用，但也引发了许多问题：

- 每个人的文档自成体系，经常有人问最新的《DCE 4.0 产品文档》在哪儿？
- 多版本、多语言文档管理成问题
- 从 wiki 导出的 html、word、pdf 表现不好
- 已发布的用户文档没有集中存放

## 期望目标

挺哥语录：

> > 🎵我们的初步目标是建立 KubeSphere 这样的多版本、多语言、动态更新的文档中心。🎶
